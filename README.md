Backend per avatar chatbot (Express). Il frontend è stato derivato da [r3f-virtual-girlfriend](https://github.com/wass08/r3f-virtual-girlfriend-frontend).

## Setup

Crea un `.env` alla radice (vedi `.env.example`): chiave **Google Cloud** per TTS/STT, credenziali **Pupau**, `VITE_API_URL`.

Scarica **Rhubarb** per il tuo OS da [rhubarb-lip-sync releases](https://github.com/DanielSWolf/rhubarb-lip-sync/releases) e metti l’eseguibile in `bin/` (es. `bin/rhubarb`).

Avvio locale:

```
npm install
npm run dev
```
